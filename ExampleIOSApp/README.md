# WaktuSolatSwift iOS Example

This is a native SwiftUI iOS app that consumes the local `WaktuSolatSwift` package.

## Open in Xcode

1. Open `WaktuSolatSwiftExample.xcodeproj`.
2. Choose an iOS Simulator.
3. Build and run the `WaktuSolatSwiftExample` target.

## What it demonstrates

- Importing `WaktuSolatSwift` from a local Swift package reference
- Calling `getPrayerTimesByZone("SGR01")`
- Calling `getPrayerTimeByDate("SGR01", date: Date())`
- Rendering simple results in SwiftUI
